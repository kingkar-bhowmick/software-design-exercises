# Softwareentwurf – Übungsrepository (Starter)

Willkommen im zentralen **Starter-Repository** für das Modul **Softwareentwurf (Wintersemester 2025/26)** im Studiengang *Bachelor Wirtschaftsinformatik*.

Dieses Repository enthält **Startcode und Aufgabenstellungen** für die Übungen.

## Ziel dieses Repositories
- Bereitstellung von Beispielcode und Aufgaben für alle Teams.
- Gemeinsame Basis für Ihre eigenen Arbeiten im Modul.
- Synchronisation neuer Aufgaben über *Upstream Pulls*.

Sie **bearbeiten keine Aufgaben direkt in diesem Repository**, sondern in Ihrem eigenen **Fork**.

Weitere Hinweise zum Vorgehen erhalten Sie in der Veranstaltung und auf dem Übungsblatt im [Moodle-Kursraum](https://moodle.hsnr.de/course/view.php?id=11909).
